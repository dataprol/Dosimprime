1.1.1.24 - 26/04/2007:
==================
  - Ao abrir o programa, ele fica na bandeija do sistema Windows(System Tray).
  - Ícone fica na bandeija do sistema(system Tray) ao clicar no X para fechar o
    programa.
  - Clicando com o botão direito do mouse sobre o ícone disponível na badeija do sistema,
    surge um menu de opções. Assim, pode-se exibir a janela, escondê-la, parar e reiniciar
    o funcionamento do programa e fechá-lo/encerrá-lo.
  - Retirada solciitação de escolha de impressora do início do programa. Agora, ele utiliza
    a impressora padrão, automaticamente, podendo alterar clicando no botão "Escolher 
    Impressora".
  - Disponibilizado no menu uma opção de alterar impressora para uso pelo programa sem 
    ter de exibir a janela para tanto.

1.1.1.8 - 24/04/2007:
==================
  - Concluído a emulação dos seguintes comandos da EPSON: 
    Dupla Largura / ESC + W + "1"
    Negrito / ESC + E,
    Desativar Negrito / ESC + "F"
    Itálico / ESC + "4"
    Desativar Itálico / ESC + "5"
    Sublinhado / ESC + "-" + CHR(1)
    Desativar Sublinhado / ESC + "-" + CHR(0)
    Condensado / CHR(15)
    Normal(10 CPI) / CHR(18)
    Ejetar / CHR(12)
    15 CPI / ESC + "g"
    Desativar 15 CPI e Elite(12 CPI) / ESC + "P"
    Dupla Impressão / ESC + "G"
    Desativar Dupla Impressão / ESC + "H"
  - OBS: Neste sistema, o comando Dupla Impressão, apenas, ativa o negrito
    não havendo diferença entre texto em negrito e com dupla impressão. O 
    comando Dupla Largura imprime os textos em tamanho duplicado mantendo a
    proporção original, diferentemente da impressão feita em impressoras
    matriciais EPSON onde o comando faz com que os caracteres fiquem
    alargados.
